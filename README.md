# 魔术计算器 (Magic Calculator)

这是一个复刻 2026 年春晚经典魔术的 Web 计算器程序。外观完美还原 iOS 原生计算器，并内置了巧妙的“时间同步”魔术逻辑。

## 体验地址

https://mc.rick216.cn/

## 功能特性

- **极致还原**：采用纯 HTML/CSS/JS 实现，界面模拟 iOS 18 风格，支持暗黑模式。
- **魔术核心**：通过特定的操作流程，使最终计算结果始终等于当前的系统时间（月日时分格式）。
- **沉浸式体验**：
  - 支持全屏模式。
  - 针对 iPhone 提供 PWA 支持（添加到主屏幕即可获得全屏体验）。
  - 动态调整数字字号。
- **零依赖**：原生代码编写，无需任何框架，运行极快。

## 魔术演示流程 (表演步骤)

1. **随机数 A**：邀请第一个观众输入一个 4 位数的随机数（例如：1234），然后点击 `+`。
2. **随机数 B**：邀请第二个观众输入一个 5 位数的随机数（例如：56789），然后再次点击 `+`。
   - *此时计算器内部已自动计算并锁定魔法差值。*
3. **随机盲按**：将手机反扣或交给观众，让他们随意按下数字键。
   - *无论观众按什么键，屏幕都会按顺序显示“目标时间 - 前两项和”的差值位，制造出乱码输入的假象。*
4. **见证奇迹**：点击 `=`，最终结果将精准显示为当前的系统时间（如 2月16日22:50 显示为 `2162250`）。

## 如何本地运行

1. 下载或克隆本仓库。
2. 在浏览器中直接打开 `index.html`，或使用本地服务器运行：
   ```bash
   npx serve .
   ```

## 手机端全屏说明 (iOS)

由于 iOS Safari 浏览器的限制，普通的网页无法直接全屏。若要达到完美表演效果：
1. 在 Safari 中打开此网页。
2. 点击分享按钮（底部的方框按钮）。
3. 选择 **“添加到主屏幕”**。
4. 从主屏幕打开“魔术计算器”，即可享受无地址栏的全屏表演。
